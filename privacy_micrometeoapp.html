Privacy Policy — MicrometeoApp

Effective date: 2025-10-03

Summary

MicrometeoApp requests access to the user's geolocation only to retrieve a weather forecast from api.open-meteo.com. Location coordinates are used solely for the API request, are not stored on our servers, and are not shared with third parties by MicrometeoApp.

Data Collected

  • Coordinates (latitude/longitude) — obtained via the browser Geolocation API after explicit user consent.
  • Forecast data — responses from https://api.open-meteo.com/v1/forecast (temperature, precipitation, wind, etc.).
  • Not collected: name, email (unless submitted by user), device identifiers, movement history.

Purpose and Use

Coordinates are used once to perform a request to Open-Meteo and to display the forecast inside the extension. We do not store coordinates on our servers and do not use them for targeting or analytics.

Transmission and Storage

Requests containing coordinates are sent directly to https://api.open-meteo.com. MicrometeoApp does not transmit user coordinates to third parties. Temporary network logs may exist on the API provider side; please consult Open-Meteo's privacy policy for details.

Extension Permissions

The extension uses minimal host permissions: https://api.open-meteo.com/*. Location permission is requested by the browser via the Geolocation prompt and is not requested silently in the background.

User Requests and Deletion

For requests to delete data or to inquire what data could have been processed, contact: micrometeoapp@gmail.com. We will respond and act in a timely manner.

Contact

Email: micrometeoapp@gmail.com

If you discover a privacy or security issue, please contact the address above.
